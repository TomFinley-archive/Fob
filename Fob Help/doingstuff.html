Doing Stuff

This section describes what the user may do, and the results of those actions, and alternate methods of accomplishing these actions if not obvious.

Start / Set New Alarm
The alarm currently displayed in the clock will be started, displaying it in the current alarm table.
Text Alarm / Set Alarm From Text
This allows the user to define an alarm textually with nothing but the keyboard. The syntax for this text as well as several examples are provided here. The action currently displayed in the Fob main window will perform when the alarm runs out of the time.

Store / Add As Preset
The alarm currently displayed in the clock will be stored as a preset alarm.
Start Selected / Start Selected Presets
This starts the selected alarms of the preset table. The user may also start a preset alarm by double clicking its row in the preset table, or by selecting it from the dock menu.
Delete
The item currently selected will be deleted, whether it be an alarm or any other item. Hitting the delete key accomplishes the same effect.
Removed Selected Alarms / Removed Selected Presets
Those alarms selected in current alarms (or presets, for "Remove Selected Presets") will be deleted.
Clear Due / Clear Due Alarms
Any alarm that has run out of time will be deleted. This is also available from the dock menu.
Rewind / Rewind Selected/Last Due Alarm
Takes any selected active alarms, and "rewinds" them so that they have the time that they originally started with. If no alarm is selected, Fob rewinds the non-deleted alarm that most recently ran out of time. If no alarm is selected and no alarm is due, nothing happens. One may also select Rewind Last Due from the dock menu to rewind the non-deleted alarm that most recently ran out of time.
Pause / Pause Selected Alarms
Selected active (currently running) alarms will be paused. You can alternatively double-click an active alarm, or select it from the dock menu.
Unpause / Unpause Selected Alarms
Reactivate the selected paused alarms. You can alternatively double-click a paused alarm, or select it from the dock menu.
Preferences
This allows the user to edit preferences.
